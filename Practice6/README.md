# Practice 6 — Python File Handling and Built-in Functions

## Structure

```
Practice6/
├── file_handling/
│   ├── read_files.py          # read(), readline(), readlines(), iteration
│   ├── write_files.py         # write, append, exclusive create modes
│   └── copy_delete_files.py   # shutil.copy/copy2, safe delete
├── directory_management/
│   ├── create_list_dirs.py    # mkdir, makedirs, listdir, walk, rglob
│   └── move_files.py          # shutil.move, copytree, rmtree
├── builtin_functions/
│   ├── map_filter_reduce.py   # map, filter, reduce, sorted, type conversions
│   └── enumerate_zip_examples.py  # enumerate, zip, any, all, range
└── README.md
```

## How to run

```bash
# From the Practice6 folder:
python file_handling/read_files.py
python file_handling/write_files.py
python file_handling/copy_delete_files.py

python directory_management/create_list_dirs.py
python directory_management/move_files.py

python builtin_functions/map_filter_reduce.py
python builtin_functions/enumerate_zip_examples.py
```
